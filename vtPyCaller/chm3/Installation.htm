Installation

System Requirements

 

Hardware : 486DX/66 MHz PC or higher,  Windows 95/98/NT4/2000/ME/XP/VISTA

Software : Virtools Dev 4.0.x or higher , and Python 2.5 or higher 

 

Installation :

- Copy the supplied vtPythonCaller.dll into your "BuildingBlocks" Directory ( ie: "c:\program files\virtools dev 4\BuildingBlocks" )

- Copy the supplied "MSVCP71.dll" and "MSVCR71.Dll" into your Virtool main directory. You don't need to overwrite you existing ones.

 

To Uninstall vtPython :

 

- Remove the "vtPythonCaller.dll" from your "BuildingBlocks" directory described in the installation section.
