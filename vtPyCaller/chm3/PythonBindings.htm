Python Bindings Reference

 

Technical Information

In this chapter we introduce to the communication ways between a Python script and virtools.

 

Building block related functions :

 

GetInVal(BuildingBlockID,IndexOfParameterInput)

Description : "GetInVal" retrieves an input parameter value from a building block.

Parameter Access Type Description
BuildingBlockID read only Integer

Specifies the unique identifier of the building block used to run this python script. In your script you just have to write  : bid

IndexOfParameterInput  read only Integer This is the index of building blocks parameter input. The type can only be a String, Integer, Float or Boolean.

 

 

 

 

 

SetOutVal(BuildingBlockID,IndexOfParameterInput)

Description : "SetOutVal" sets an output parameter value of a building block.

Parameter Access Type Description
BuildingBlockID read only Integer

Specifies the unique identifier of the building block used to run this python script. In your script you just have to write  : bid

IndexOfParameterInput  read only Integer This is the index of building blocks parameter output. The type can only be a String, Integer, Float or Boolean. For the case a Python function returns an array then it's converted to a string where the values are seperated by a comma. 

 

 

 

 

 

 

ActivateOut(BuildingBlockID,IndexOfParameterInput,State)

Description : "SetOutVal" sets an output parameter value of a building block.

Parameter Access Type Description
BuildingBlockID read only Integer

Specifies the unique identifier of the building block used to run this python script. In your script you just have to write  : bid

IndexOfParameterInput  read only Integer This is the index of building blocks parameter output. The type can only be a String, Integer, Float or Boolean. For the case a Python function returns an array then it's converted to a string where the values are seperated by a comma. 
State read only Integer The output triggers state.

 

 

 

 

 

 

IsInputActive(BuildingBlockID,IndexOfInput)

Description : "IsInputActive" return the state of an building block input trigger.

Parameter Access Type Description
BuildingBlockID read only Integer

Specifies the unique identifier of the building block used to run this python script. In your script you just have to write  : bid

IndexOfInput  read only Integer This is the index of building blocks trigger input.

# import the virtools module :  

import vt

def multiply():

   #We retrieve our variable "a" from the building block and we assume the type of the parameter input is set to Integer.
   #This can be done with vt.GetInVal(bid,IndexOfParameterInput). "bid" is an automatically created variable when the 
   #building blocks starts. It is an unique identifier for the building block for intern processing.Just write "bid" !
   a  = vt.GetInVal(bid,3)

   # Now we retrieve variable "b" : 
   b  = vt.GetInVal(bid,4) 


   c = 0
   for i in range(0, a):
     c = c + b

   #When the loop is finish, we store the result in the building blocks parameter output : 
   vt.SetOutVal(bid,0,c)

   #And we trigger an output we created on the building block :

   vt.ActivateOut(bid,0,1)

   #vt.ActivateOut(bid,2,1) means : activate output trigger number 2 wherer the last "1" sets the state to true !
