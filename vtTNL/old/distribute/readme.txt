1.0 General Notes 
  1.1 Installation      
      - copy from BuildingBlocks\vtTNL.dll to your dev\BuildingBlocks directory
      - for testing of your application I recommend :
        - copy the entire Dev-YOUR-VERSION directory into your dev directory !
        - call vtNetServer.bat(a custom player in console mode) to start a cmo driven server or         
          start "xConsoleServer.exe" (virtools independent server module !)

2.0 Usage 
	- start the server : xConsoleServer.exe or the embedded one !
	- create a session ( with password )
	- join to this session
               - now you can send messages and attach parameters of the type :
                  - Vector 2/3/4 - D 
                  - String (256 chars per message !) 
                  - int /float

      Documentation in progress ! ;-)
